Set up git:
git --version
git config --global user.name
git config --global user.email
git config --global init.defaultBranch (one time for default set from master to main)

After creating project:
git init (have to do only one time)
ls -a or lsd -a -tree
rm -rf .git (completly delete git from the project)
git status
git add . (can add individully like: git add index.html)
git commit -m "commit like a comment"
git log (shows the commits you have done previously) (??press q to escape??)

Push an existing repository from the command line:
git remote add origin https://github.com/hasibur049/task-tracker.git
git branch -M main
git push -u origin main
token (get token from .env file)
git pull (from remote repo)

Shortcut of commit :
git commit -am "dont have to do add . manunally for this shortcut"
git add index.html && git commit -m "added new feature"
